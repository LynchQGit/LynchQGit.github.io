    类继承使用¶

    尝试利用继承关系来找到object类 

    >>> "".__class__.__bases__   #(<class 'object'>,)
(<class 'object'>,)

    >>> [].__class__.__bases__
(<class 'object'>,)
>>> {}.__class__.__bases__
(<class 'object'>,)
